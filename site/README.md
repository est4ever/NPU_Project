# AcouLM Website

Static product + documentation website for AcouLM, built with Vite + React + TypeScript + Tailwind.

## Local development

```powershell
cd site
npm install
npm run dev
```

## Production build

```powershell
cd site
npm run build
npm run preview
```

## Deployment

- Default target: GitHub Pages via `.github/workflows/pages.yml`
- Public URL: `https://est4ever.github.io/AcouLM/`
- Also compatible with Vercel free subdomain deploys
